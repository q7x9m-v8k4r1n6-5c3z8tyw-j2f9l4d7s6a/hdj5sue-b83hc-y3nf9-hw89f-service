# Documentation

This folder stores product and engineering documentation for OVCMOVE.

Recommended sections:

- `architecture/`: system design, Clean Architecture rules, data flow, diagrams
- `api/`: endpoint contracts, request/response examples, error model
- `deployment/`: Docker, environment variables, release process, CI/CD notes
- `operations/`: monitoring, logging, incident runbooks, database operations
- `decisions/`: ADRs for technical decisions
